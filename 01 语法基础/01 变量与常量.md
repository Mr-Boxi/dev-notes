# 变量与常量

## 一 变量

（1）变量的定义
```go
go代码:
    var a int
    a  := 127   //仅在函数中使用
    var (       //批量声明
      a int
      b byte
      c float32
    )
```
```rust
rust代码：
    let a: i8 = 1;
    let x = 1;  //等价于  let  x: i32 = 1;
```

## 二 常量
(1) 常量的定义
```go
go:
    const Red int = 6
```
```rust
rust:
    const MAX_NUM: u32 = 1024;
```
