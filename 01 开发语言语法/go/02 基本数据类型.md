# 基本数据类型

## go
    整型：
        int8       uint8
        int16      uint16
        int32      uint32
        int64      uint64
    浮点型：
        float32
        float64
    布尔类型：
        bool
    字符类型：
        byte
        rune
    字符串：
        string
## rust
    整型：
        i8          u8
        i16         u16
        i32         u32
        i64         u64      
        i128        u128
        isize(arch) usize
    浮点型：
        f32
        f64
    布尔类型
        bool
    字符类型：
        由单引号定义，如 let a = 'u';
    字符串：
        string (与go不同，后面单独记录)
